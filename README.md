# Atelier Rabab 🎨

Plateforme e-commerce de luxe pour l'artisanat fait main, spécialisée dans les créations en résine et plâtre.

## 🛠 Technologies

Ce projet est développé avec une stack moderne :

- **Framework:** React.js (Vite)
- **Langage:** TypeScript
- **Styling:** Tailwind CSS
- **Animations:** Framer Motion
- **UI Components:** Shadcn UI
- **Backend/Database:** Supabase
- **Icons:** Lucide React

## 🚀 Fonctionnalités

- **Système Multilingue:** Support complet de l'Arabe et du Français.
- **Expérience Shopping:** Panier dynamique et gestion des variantes de couleurs.
- **Commande WhatsApp:** Finalisation des commandes simplifiée via une messagerie directe.
- **Service Sur-Mesure:** Formulaire dédié pour les créations personnalisées avec envoi d'images d'inspiration.
- **Interface Adaptive:** Design responsive optimisé pour mobile et desktop.
